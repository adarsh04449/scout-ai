# Startup Landing Page Screenshots

Place your startup landing page screenshots in this directory.

## Required Images

Add the following image files (PNG or JPG format, recommended size: 1200x800px or larger):

- `stripe.png` - Stripe landing page screenshot
- `notion.png` - Notion landing page screenshot
- `linear.png` - Linear landing page screenshot
- `vercel.png` - Vercel landing page screenshot
- `figma.png` - Figma landing page screenshot
- `anthropic.png` - Anthropic landing page screenshot
- `replit.png` - Replit landing page screenshot
- `v0.png` - V0 landing page screenshot
- `clerk.png` - Clerk landing page screenshot

## How to Get Screenshots

### Option 1: Manual Screenshots
1. Visit each startup's website
2. Take a full-page screenshot using:
   - **Chrome/Edge**: Use browser extensions like "Full Page Screen Capture"
   - **Firefox**: Use built-in screenshot tool (Shift+F2, then `screenshot --fullpage`)
   - **Safari**: Use Cmd+Shift+4, then Space to capture window

### Option 2: Online Tools
- Use services like:
  - [Screenshot.guru](https://screenshot.guru) - Enter URL, get screenshot
  - [Page2Images](https://www.page2images.com) - API for screenshots
  - [BrowserStack](https://www.browserstack.com/screenshots) - Cross-browser screenshots

### Option 3: Command Line (if you have Node.js)
```bash
# Install puppeteer
npm install -g puppeteer

# Or use a screenshot service API
```

## Image Requirements

- **Format**: PNG or JPG
- **Recommended Size**: 1200x800px or larger (will be scaled down)
- **Aspect Ratio**: 3:2 or 16:10 works best
- **File Size**: Keep under 500KB per image for performance

## Naming Convention

Files should be named exactly as listed above (lowercase, with .png extension).

Example: `stripe.png`, `notion.png`, etc.
